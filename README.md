# Maze generating and solving game

## Objective
<p align="justify">
The objective of this project is to make an application that allows the user to make their own maze and to solve them. 
</p>

## How to make a maze
<p align="justify">
  
1. Run Main.java then click on "Create a maze".  

2. Choose an algorithm among the 4 displayed.  
   
3. Input row and column specifications, then click on "submit".  

4. Click on "Generate" to create a maze. the walls of the maze are represented as black tiles while the traversable tiles are shown in white.  

5. The generated maze can be edited by clicking on the tiles of the maze preview. Clicking on a black tile turns it white vice-versa.  

6) The maze can be saved by clicking on "save". If a maze the newly created maze has the same name as an existing one, the existing one will be replaced by the new maze.  
</p>

## To solve a maze
<p align="justify">

1. Run Main.java then click on "Solve a maze".  

2. Go on "option" and click on "select maze" and choose the maze you want.  

3. The goal of the game is to connect the two yellow tiles by a green trail by clicking on the tiles that are not black.  

4. The answer can be viewed by clicking on "See solution"and then by selecting an algorithm of your choice. "BFS" is the recommended algorithm of choice as it gives the shortest path in the shortest amount of time.  
</p>
